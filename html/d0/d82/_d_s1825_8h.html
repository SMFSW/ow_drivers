◆ DS1825_cmd

enum DS1825_cmd

Commands enum for DS1825.

Note
Unused
Enumerator
DS1825__CONVERT_T 

Initiates temperature conversion.

DS1825__WRITE_SCRATCHPAD 

Writes data into scratchpad bytes 2, 3, and 4 (Th, Tl, and configuration registers)

DS1825__COPY_SCRATCHPAD 

Copies Th, Tl, and configuration register data from the scratchpad to EEPROM.

DS1825__READ_SCRATCHPAD 

Reads the entire scratchpad including the CRC byte.

DS1825__RECALL 

Recalls Th, Tl, and configuration register data from EEPROM to the scratchpad.

DS1825__ALARM_SEARCH 

This command allows the master device to determine if any DS1825 experienced an alarm condition during the most recent temperature conversion.
