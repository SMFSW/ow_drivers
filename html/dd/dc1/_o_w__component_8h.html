◆ OW_slave_get_power_supply()

FctERR OW_slave_get_power_supply ( OW_slave_t * pSlave,
bool *const pBusPower )

OneWire Salve get power supply source.

Note
May be useful to keep bus as busy during a copy scratchpad command or during a conversion (line should be held high, no other transaction allowed on bus)
Warning
Use only if device supports the command (meaning it can be powered by power or bus), otherwise result will be wrong and irrelevant
Parameters
[in]pSlave- pointer to OW slave instance
[in,out]pBusPower- pointer to bus power variable result
Returns
FctERR - Error code
